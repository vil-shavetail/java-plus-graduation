## Групповой учебный проект Java. ФЗ №17. 
### Третий этап: Реализация Комментариев.

Добавлена функциональность добавления, удаления комментариев к событиям. А также функционал получения комментариев конкретного события.
#### Реализованы три эндпоинта. Два приватных и один публичный.

#### Приватные эндпоинты:
1) Добавление комментария.
2) Удаление комментария.
Добавить и удалить комментарии можно только у опубликованного события, пользователем, который осуществлял запрос на участие в данном событии и запрос был принят автоматически, либо был одобрен инициатором события.
В случае если описанные выше условия выполняются, комментарий успешно добавляется/удаляется. В противном случае генерируется исключение, которое обрабатывается.

#### Публичный эндпоинт:
1) Эндпоинт просмотра комментариев к событию носит публичный характер. Получение перечня комментариев события, осуществляется по идентификатору события!
При запросе комментариев у события, возвращается список комментариев. Если у события не комментариев, возвращается пустой список.
